=head2 new

B<new(PARENT, %OPTIONS)>

Creates a new Slider object;
can also be called as PARENT->AddSlider(%OPTIONS).

Class specific B<%OPTIONS> are:

  -tooltip => Win32::GUI::Tooltip
  -vertical => 0/1
  -aligntop => 0/1
  -alignleft => 0/1
  -noticks => 0/1
  -nothumb => 0/1
  -selrange => 0/1
  -autoticks => 0/1
  -both => 0/1
  -fixedlength => 0/1
